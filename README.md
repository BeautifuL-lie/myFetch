
# MyFetch

A very simple neofetch/fastfetch-like program written in C.




## Installation

To compile the program, first run this in terminal:

```bash
make
```
Then install it, with:

```
sudo make install
```
Then run it with:
```
myfetch
```
..Or, if you don't want to install it. After compiling the program, run:
```
make run
```

## Uninstall
To uninstall it, run:
```
sudo make uninstall
```
